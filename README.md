# Аналитика будущего рынка труда в эпоху ИИ (2020–2026)

Этот проект посвящён анализу глобальных изменений на рынке труда под влиянием искусственного интеллекта.  
Используется датасет **Future of Work in the Age of AI (2020–2026)** с Kaggle, содержащий более 15 000 строк данных о профессиях, отраслях, странах, уровне автоматизации, рисках замещения, зарплатах и динамике изменений.

Цель проекта — выявить тенденции трансформации рынка труда, определить наиболее уязвимые профессии, оценить влияние ИИ на отрасли и страны, а также выделить ключевые навыки будущего.

---

## Структура проекта

Проект организован по принципам аналитики данных и включает следующие разделы:

- `sql/` — SQL-скрипты для создания таблиц, загрузки данных и аналитических запросов.
- `python/` — скрипты для ETL-процессов (загрузка и очистка данных), а также файлы с EDA (исследовательским анализом данных).
- `powerbi/` — итоговый интерактивный дашборд Power BI, визуализирующий ключевые метрики и результаты анализа.
- `docs/` — документация проекта: словарь данных, архитектура, описание витрин, гипотезы и план анализа.
- `data/` — ссылки на исходный датасет Kaggle (файл не хранится в репозитории из-за ограничений лицензии).

## Архитектура проекта

Проект построен по принципу аналитического конвейера (data pipeline), где каждый этап отвечает за свою часть обработки данных:

### 1. **CSV → Python (ETL)**  
- загрузка исходного файла;
- очистка данных (дубликаты, пропуски, типы данных);
- нормализация и подготовка к загрузке в БД.

### 2. **Python → PostgreSQL**  
- создание таблиц;
- загрузка очищенных данных в базу;
- формирование витрин для анализа.

### 3. **PostgreSQL → Python (EDA)**  
- исследовательский анализ данных (EDA);
- визуализации (seaborn, matplotlib);
- статистические тесты (scipy);
- выявление закономерностей:
  - профессии с высоким риском автоматизации  
  - отрасли, наиболее подверженные влиянию ИИ  
  - динамика зарплат  
  - анализ навыков будущего  

### 4. **PostgreSQL → Power BI**  
- подключение к витринам;
- построение интерактивного дашборда;
- визуализация ключевых метрик:
  - риск автоматизации  
  - влияние ИИ  
  - зарплаты  
  - отрасли  
  - страны  
  - навыки

# Итоговый вывод по проекту «Аналитика будущего рынка труда под влиянием ИИ»

Проект объединяет SQL‑анализ, Python‑исследование данных, статистические тесты и Power BI‑визуализацию, чтобы оценить влияние искусственного интеллекта на рынок труда, профессии и зарплаты.


## 1. SQL‑анализ: структура рынка труда

SQL‑запросы показали:

- Профессии с высоким риском автоматизации распределены **неравномерно**, но нет индустрий, полностью защищённых от ИИ.
- Зарплатные изменения **не коррелируют напрямую** с уровнем риска — есть профессии с высоким риском и растущими зарплатами.
- Структура рынка труда смещается в сторону профессий, требующих **когнитивных навыков, анализа данных, коммуникации и творчества**.

**Главный вывод SQL:**  
ИИ влияет на рынок труда **точечно**, а не равномерно по индустриям.

## 3. Кластеризация профессий (PCA + KMeans)

Кластеры профессий **значимо различаются**:

- по уровню риска автоматизации  
- по ожидаемым изменениям зарплат  

Это подтверждено статистически (p-value = 0.0000).

**Главный вывод кластеризации:**  
Профессии группируются в **устойчивые сегменты**, отличающиеся по влиянию ИИ.

## 4. Статистический анализ (Shapiro, Kruskal, Spearman)

### Нормальность
- `automation_risk_percent` → не нормальное  
- `ai_replacement_score` → не нормальное  
- `salary_change_percent` → почти нормальное  

### Различия между индустриями
- p = 0.276 → **различий нет**

### Различия между кластерами
- зарплаты: p = 0.0000 → **значимые различия**
- риск: p = 0.0000 → **значимые различия**

### Корреляции
- `automation_risk_percent` ↔ `ai_replacement_score`: **rho = 0.969 (очень сильная связь)**
- Остальные корреляции → **не значимы**

**Главный вывод статистики:**  
Риск автоматизации и AI‑replacement — фактически **одна и та же метрика**,  
а зарплаты **не зависят** от риска автоматизации.

## 5. Общий итог проекта

1. **ИИ не влияет одинаково на все индустрии** — различий между индустриями нет.  
2. **Кластеры профессий различаются сильно**, что подтверждает сегментацию рынка труда.  
3. **Риск автоматизации не связан с падением зарплат** — миф о «ИИ снижает зарплаты» не подтверждён.  
4. **AI Replacement Score и Automation Risk почти идентичны** (rho ≈ 0.97).  
5. **Skill gap не объясняет риск автоматизации** — влияние ИИ сложнее, чем просто нехватка навыков.  
6. **Рынок труда становится более поляризованным**: одни профессии исчезают, другие растут.  
7. **SQL + Python + Power BI дают целостную картину**, где каждая технология усиливает другую.

# Финальный вывод

ИИ **не уничтожает рынок труда**, а **перестраивает его**.  
Профессии не исчезают массово — они **перераспределяются**.  
Ключевым становится не «бороться с ИИ», а **адаптироваться к новым требованиям**.

Проект показывает, что:

- влияние ИИ **не хаотично**,  
- оно **структурировано**,  
- и его можно **измерить, сегментировать и визуализировать**.
